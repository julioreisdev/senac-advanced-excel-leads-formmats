Desafio: Higienização e Análise de Dados

Nossa Missão

Bem-vindo(a) à equipe de migração de dados da ConnectaSolutions!

Nossa empresa está passando por uma grande modernização e vamos aposentar nosso sistema antigo de contatos. Estamos implementando um novo e poderoso sistema de CRM chamado "Nexus". O problema? O "Nexus" é muito rigoroso com a qualidade dos dados que ele aceita.

Ao longo do último ano, nossos vendedores cadastraram os contatos (leads) de qualquer jeito. Agora, temos uma base de dados com nomes e telefones em formatos variados, com erros, em branco e até com números inválidos. Sua missão, como especialista em dados, é limpar essa base para que possamos importá-la para o "Nexus" sem erros.

O novo sistema só aceita números de telefone no formato padrão brasileiro: +55 (XX) X XXXX-XXXX.

Suas Tarefas

Esta atividade deverá ser realizada em duplas. Colaborem para encontrar as melhores soluções!

  1. Baixe a base de dados de leads clicando no botão abaixo.
  2. Abra o arquivo CSV no Excel e analise os problemas existentes nos nomes e telefones.
  3. Limpe e padronize os nomes, removendo espaços extras e ajustando a capitalização (letras maiúsculas/minúsculas).
  4. Trate os números de telefone, identificando e lidando com os que estão em branco ou são inválidos (muito curtos).
  5. Para os números válidos mas mal formatados, use fórmulas para convertê-los ao formato padrão exigido pelo sistema "Nexus".
  6. Crie uma pequena análise respondendo: Qual o percentual de números que estavam incorretos (incluindo em branco e inválidos) na base de dados original?
  7. Envie o arquivo final (Excel com os dados corrigidos e a análise) para o e-mail: juliopais@pi.senac.br.
